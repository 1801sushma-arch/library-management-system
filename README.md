# library-management-system(Python OOP)

## 📌 Description
A simple Python project that simulates a library using Object-Oriented Programming.

## 🚀 Features
- View available books
- Borrow a book
- Return a book

## 🧠 Concepts Used
- Classes & Objects
- Methods
- Lists
- Basic input handling (strings)

## ▶️ How to Run
python library-management-system.py

## 💡 What I Learned
- Designing a system using classes
- Managing state (available books) inside an object
- Handling user-like actions (borrow/return)
- Writing clean, readable methods

## 🔧 Future Improvements
- Add file storage (save/load books)
- Add multiple users
- Add search functionality
- Prevent duplicate entries robustly
